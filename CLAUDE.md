# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## What This Is

A forked and customized instance of [posquit0/Awesome-CV](https://github.com/posquit0/Awesome-CV) — a LaTeX template for generating a PDF resume. **The `.tex` files in `sudeep/` are auto-generated** from TypeScript data; edit the source data, not these files directly.

## Build Commands

```bash
# Compile PDF (requires TeX Live + LuaLaTeX installed locally)
make sudeep

# Compile via Docker (recommended — reproducible, no local TeX install needed)
docker build -t sudeep-awesome-cv .
docker run --rm --user $(id -u):$(id -g) -e TEXMFVAR=/doc/.cache \
  -i -w "/doc" -v "$PWD":/doc sudeep-awesome-cv make sudeep
```

Output: `sudeep/Sudeep_Resume.pdf`

## Data Flow

**Do not edit `sudeep/` files manually.** They are generated by the sync pipeline:

```
../resume/src/data/sudeep-cv.ts   ← edit here
        ↓  npm run sync:to-latex (run from ../resume/)
sudeep/main.tex
sudeep/subtopics/{summary,experience,skills,projects,education}.tex
        ↓  make sudeep
sudeep/Sudeep_Resume.pdf
```

To update resume content:
1. Edit `../resume/src/data/sudeep-cv.ts`
2. `cd ../resume && npm run sync:to-latex`
3. `make sudeep` (or use Docker command above)

## File Structure

```
awesome-cv.cls              # LaTeX class — defines all CV commands and layout
Makefile                    # Build rules (target: sudeep)
Dockerfile                  # TeX Live image with Roboto + Source Sans fonts
sudeep/
  main.tex                  # Document header (personal info, margins, color scheme)
  awesome-cv.cls -> ../     # Symlink to parent class file
  fonts/                    # Roboto + FontAwesome TTF files
  subtopics/
    summary.tex             # \cvparagraph
    experience.tex          # \cventry blocks
    skills.tex              # \cvskill blocks
    projects.tex            # \cventry blocks (visible: false entries are commented out)
    education.tex           # \cventry blocks
  Sudeep_Resume.pdf         # Compiled output (gitignored)
```

## Key LaTeX Commands (from awesome-cv.cls)

```latex
\cvsection{Name}                          % Section heading
\begin{cventries}
  \cventry{Title}{Org}{Location}{Dates}{description}
\end{cventries}

\begin{cvskills}
  \cvskill{Category}{skill1, skill2, ...}
\end{cvskills}

\cvparagraph{text}                        % Prose paragraph (used for summary)
```

## CI/CD

GitHub Actions (`.github/workflows/main.yml`) automatically compiles the PDF and uploads it as an artifact on every push and PR. It installs Roboto and Adobe Source Sans 3 fonts, then runs `make`.
